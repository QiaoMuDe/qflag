flowchart TD
    Start([开始]) --> Parse[ParseAndRoute]
    
    Parse --> CheckParsed{是否已解析?}
    CheckParsed -->|是| HandleBuiltin[处理内置标志]
    CheckParsed -->|否| ParseOnly[ParseOnly 解析参数]
    
    ParseOnly --> LoadEnv[加载环境变量]
    LoadEnv --> ValidateGroups[验证互斥组和必需组]
    ValidateGroups --> CheckSubCmd{是否有子命令?}
    
    CheckSubCmd -->|是| ParseSubCmd[递归解析子命令]
    CheckSubCmd -->|否| SetParsed[设置已解析状态]
    
    ParseSubCmd --> CheckParsed
    
    SetParsed --> HandleBuiltin
    HandleBuiltin --> CheckBuiltin{是否有内置标志?}
    
    CheckBuiltin -->|是| ExecuteBuiltin[执行内置标志处理]
    CheckBuiltin -->|否| CheckRunFunc{是否有运行函数?}
    
    ExecuteBuiltin --> Exit([退出])
    
    CheckRunFunc -->|是| RunFunc[执行Run函数]
    CheckRunFunc -->|否| Exit
    
    RunFunc --> CheckError{执行是否成功?}
    CheckError -->|是| Exit
    CheckError -->|否| HandleError[处理错误]
    
    HandleError --> Exit
